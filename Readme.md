# Todo Card Update
Here are some functionality that have been added to the card

- Edit Functionality which includes being able to edit informations on the todo component and save
- An Edit form to pass in new values for the card

## Design Decisions
Little changes were made in the design to maintain responsiveness. There were slight changes in the color used on some tags and status button

## Limitation
The focus state for the status badge is not visible but might be discovered

## Accessibility Note
The accessibility implemented is mainly the keyboard navigable contents. Where the user can navigate through the elements to make changes to the card if needed.
